# Termux YT-DLP Setup

Skrip otomatis untuk menginstall yt-dlp dan ffmpeg di Termux dengan fitur menu kualitas dan download playlist.

### Cara Install
Salin dan tempel perintah ini di terminal Termux Anda:

```bash
curl -L [https://raw.githubusercontent.com/muizka/termux-yt/refs/heads/main/install.sh](https://raw.githubusercontent.com/muizka/termux-yt/refs/heads/main/install.sh) | bash
